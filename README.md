## Project Details

### Project Title - EazyStore

 EazyStore is a small e-commerce app for selling vinyl stickers. The repository contains two folders: easystore-UI (a React 19 + Vite + TailwindCSS storefront) and eazystore-api (an Express + Mongoose REST API backed by an in-process mongodb-memory-server). This document walks through every feature of both halves and shows the exact code that implements it.
